API Documentation

üìö Books Endpoints

GET /api/books
Retrieve all books. Supports filtering by availability, genre, and author.
Query Parameters: available (boolean), genre (string), author (string)
GET /api/books/:id
Retrieve a specific book by ID.
POST /api/books
Create a new book entry.
Required: title, author, isbn
Optional: genre, publishedYear, available
PUT /api/books/:id
Update an existing book.
DELETE /api/books/:id
Delete a book from the library.
POST /api/books/:id/borrow
Borrow a book for a specific user.
Body: { "borrowerId": number }
POST /api/books/:id/return
Return a borrowed book.

‚úçÔ∏è Authors Endpoints

GET /api/authors
Retrieve all authors.
GET /api/authors/:id
Retrieve a specific author with their books.
POST /api/authors
Create a new author.
Required: name
Optional: biography, birthYear, nationality
PUT /api/authors/:id
Update an existing author.
DELETE /api/authors/:id
Delete an author.

üë• Borrowers Endpoints

GET /api/borrowers
Retrieve all borrowers.
GET /api/borrowers/:id
Retrieve a specific borrower with their borrowed books.
POST /api/borrowers
Create a new borrower.
Required: name, email
Optional: membershipDate
PUT /api/borrowers/:id
Update an existing borrower.
DELETE /api/borrowers/:id
Delete a borrower (only if no active loans).
